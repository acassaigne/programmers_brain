# Atelier lecture de code


Dans nos écoles, nous avons appris à lire avant de savoir écrire.
D'après les méthodes d'enseignement: Comprendre un texte, c’est se faire une représentation mentale cohérente qui intègre toutes les informations du texte". Ces principes s'appliquent tout aussi bien à la compréhension de n'importe quel code logiciel.
Car nous avons tous fait l'expérience de lire du code qui ne nous appartenais pas ou même nos propres productions.

Nos constats sont les suivants:

1. Il est souvent difficile et compliqué de lire ou de relire un code source.
2. C'est  une activité pénible et démotivante pour grand nombre des développeurs.
3. Et si la raison ne tenait qu'a notre méconnaissance des mécanismes de fonctionnement de notre cerveau ?

Venez explorer, expérimenter et apprendre comment fonctionne notre cerveau lors d'un atelier de lecture d'un code source
Et tirez profit de ces nouvelles connaissances pour améliorer vos capacités de lecture ainsi que la lisibilité de votre code source. 

L'atelier se base sur les exercices proposés par Felienne Hermans dans son ouvrage "The Programmer's Brain".

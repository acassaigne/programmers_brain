# Atelier lecture de code


Dans nos écoles, nous avons appris à lire avant de savoir écrire.
D'après les méthodes d'enseignement: Comprendre un texte, c’est se faire une représentation mentale cohérente qui intègre toutes les informations du texte". Ces principes s'appliquent tout aussi bien à la compréhension de n'importe quel code logiciel.

1. Il est souvent difficile et compliqué de lire ou de relire un code source.
2. C'est donc une activité pénible et démotivante pour le plus grand nombre des développeurs.
3. Et si la raison ne tenait qu'a notre méconnaissance des mécanismes de fonctionnement de notre cerveau ?
4. Venez explorer, expérimenter et apprendre comment fonctionne notre cerveau lors de la lecture d'un code source et tirer profit de ces nouvelles connaissances pour améliorer vos capacités de lecture ainsi que la lisibilité de votre code source. (L'atelier se base sur les exercices proposés par Felienne Hermans dans son ouvrage Programmer's Brain).

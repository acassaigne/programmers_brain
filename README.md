# Atelier lecture de code


1. Il est souvent difficile et compliqué de lire ou de relire un code source.
2. C'est donc une activité pénible et démotivante pour le plus grand nombre des développeurs.
3. Et si la raison ne tenait qu'a notre méconnaissance des mécanismes de fonctionnement de notre cerveau ?
4. Venez explorer, expérimenter et apprendre comment fonctionne notre cerveau lors de la lecture d'un code source et tirer profit de ces nouvelles connaissances pour améliorer la lisibilité de votre code source. (L'atelier se base sur les exercices proposés par Felienne Hermans dans son ouvrage Programmer's Brain).
